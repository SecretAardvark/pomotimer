pomotimer root command 
Start command: 
    Starts a pomodoro timer. 
    --task or -t flag:
        indicate which task you are timing, defaults to none. 
    --interval or -int
         set the number of minutes you to time as an int, defualts to 25 minutes. 
Pause command: 
    Pauses the currently running timer. Should only work 
    if there is a timer running. 
Unpause command: 
    Unpauses the currently paused timer. Should only work 
    if the timer is NOT running. 
Add command:
    Adds a focus task to the task DB. 
    task argument: 
        The focus task you are adding to the DB. 
Show command: 
    Shows the stats
